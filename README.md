# AIOps & IA na Engenharia de Cloud — Portfolio Técnico

Repositório de exercícios práticos do programa de especialização em **AIOps e Inteligência Artificial aplicados à Engenharia de Cloud**. Documentação de padrões, frameworks e técnicas avançadas em **DevOps, SRE e Automação de Infraestrutura** com IA.

Este repositório demonstra a aplicação de frameworks de engenharia de prompt, automação e arquitetura cloud em contextos reais de operação. Cada exercício articula boas práticas SRE, diagnóstico de incidentes, otimização de custos e modernização de infraestrutura.


## 🗂️ Estrutura

```
aiops-tasks/
├── Engenharia de Prompt e Contexto/          # Frameworks de prompt engineering aplicados a SRE/Cloud
│   ├── 01 - Dockerfile/                      # RTF: Container security e best practices
│   ├── 02 - Script de Backup/                # TAG: Automação resiliente com Bash
│   ├── 03 - Relatorio FinOps/                # RTF: Análise de custos AWS
│   ├── 04 - Relatorio Transactions/          # TAG: SQL avançado e data analytics
│   ├── 05 - Modernizacao Deployment/         # TAG: Kubernetes manifests production-ready
│   ├── 06 - Modulo Terraform/                # CARE: IaC modular e reutilizável
│   ├── 07 - Runbook de Alertas/              # BAB: Resposta a incidentes estruturada
│   ├── 08 - Postmortem Tecnico/              # BAB + RISE: Análise de falhas com metodologia
│   └── README.md
└── README.md
```
